# final-project-kanban-board
student task manager

A simple project created to demonstrate Agile/Scrum practices including user stories, acceptance criteria, sprint planning, issue templates, and Kanban workflow.
Main Features
Add student tasks
Mark tasks completed
View pending tasks
Organize academic work efficiently
Tools Used
GitHub Issues
GitHub Projects (Kanban Board)
User Story Template
Milestones (Sprint 1)
Labels & Estimates
Burndown Chart
